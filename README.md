## About

A modern, single-page corporate landing page showcasing Nusantara Palm Industries (NPI), Indonesia's leading sustainable palm oil exporter. This project demonstrates AI-assisted frontend development using MiniMax AI and the [Frontend Design Skill](https://claude-plugins.dev/skills/@anthropics/claude-code/frontend-design) to create production-grade interfaces.

## Project Overview

Single-page corporate landing page featuring multiple scrollable sections:

- Sustainability Section
- Mission Section
- Geographic Section
- Product Section
- Contact Section

Modern, responsive design with smooth animations and a professional industrial aesthetic.
