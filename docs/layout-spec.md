# Nusantara Palm Industries — Layout Specifications

**Document Version:** 1.0
**Last Updated:** January 2026
**Color Palette:** Nusantara Earth (Recommended)
**Visual Style:** Modern Industrial

---

## Overview

Three detailed layout specifications for the NPI landing page. Each offers a distinct visual approach while maintaining UX principles and brand alignment. Choose based on desired aesthetic and portfolio demonstration goals.

---

## Layout 1: Classic Split

**Approach:** Consistent 50/50 split throughout all content sections, creating predictable rhythm and maximum readability.

### Section-by-Section Specifications

#### 1. Hero Section
- **Type:** Full-width (not split)
- **Height:** 100vh (full viewport)
- **Background:** Burnt Sienna (#c45b28) with dark gradient overlay (opacity 60-80%)
- **Content Position:** Center-aligned, vertically centered
- **Elements:**
  - Logo: Top-left, 40px height, white
  - Nav: Top-right hamburger icon, white
  - Headline: H1, 48-64px, white, bold, centered
  - Tagline: Body large, 20-24px, white/light cream, centered, max-width 600px
  - CTA Button: Centered below tagline, 48px height, 24px padding, gold accent (#d4a24c), dark text
- **Spacing:** 80px top/bottom padding minimum
- **Responsive:** Mobile: H1 32px, Tagline 16px, CTA full-width

#### 2. Key Statistics Section
- **Type:** Split (50/50), viewport-height (min 100vh)
- **Left Panel:**
  - Width: 50%
  - Background: Dark Coffee (#2c2416)
  - Color: Warm Cream (#f5f0e8) text
  - Content: Four stat blocks stacked vertically
  - Stat Block: Large number (64-80px), label (16-18px), unit (14px)
  - Spacing: 60px padding, 40px gap between stats
  - Border: 4px left border in Gold (#d4a24c)
- **Right Panel:**
  - Width: 50%
  - Background: Warm Cream (#f5f0e8)
  - Color: Dark Coffee (#2c2416) text
  - Content: Context paragraph, max 120 words
  - Typography: Body text 18-20px, line-height 1.6
  - Positioning: Centered vertically, 80px side padding
- **Responsive:** Mobile: Stacked, left panel top (40vh), right panel bottom (60vh)

#### 3. Sustainability Section
- **Type:** Split (50/50), viewport-height
- **Left Panel (Content):**
  - Width: 50%
  - Background: Warm Cream (#f5f0e8)
  - Color: Dark Coffee (#2c2416) text
  - Content:
    - Section Title: H2, Deep Teal (#2a5c55), 36-42px
    - Body: Paragraph, 18px, max 150 words
    - Certifications: Row of 2 badge logos (RSPO, ISCC)
    - Badge Size: 80-100px width each, 40px height
    - Spacing: 60px padding
- **Right Panel (Visual):**
  - Width: 50%
  - Background: Image with overlay
  - Image: Sustainable plantation or certification imagery
  - Overlay: Deep Teal (#2a5c55) at 40% opacity
  - Content: Optional text overlay on image
- **Responsive:** Mobile: Stacked, image on top or bottom based on design preference

#### 4. Mission & Vision Section
- **Type:** Split (50/50), viewport-height
- **Left Panel (Visual):**
  - Width: 50%
  - Background: Image with Dark Coffee overlay
  - Image: Sunrise over plantation or community impact
  - Overlay: #2c2416 at 30% opacity
- **Right Panel (Content):**
  - Width: 50%
  - Background: Warm Cream (#f5f0e8)
  - Color: Dark Coffee (#2c2416) text
  - Content Order:
    - Section Title: H2, Burnt Sienna (#c45b28), 36-42px
    - Mission: Blockquote style, 20-22px italic, left border 4px gold
    - Vision: Similar to mission, distinct visual separation
    - Values: 5 items with icons, 24px icons, 16px text
    - Icon Style: Simple line or filled, consistent across all 5
    - Spacing: 60px padding, 30px gap between elements
- **Responsive:** Mobile: Stacked, visual on top

#### 5. Geographic Presence Section
- **Type:** Split (50/50), viewport-height
- **Left Panel (Content):**
  - Width: 50%
  - Background: Deep Teal (#2a5c55)
  - Color: Warm Cream (#f5f0e8) text
  - Content:
    - Section Title: H2, white, 36-42px
    - Location Groups: 3 clusters (HQ/Office, Plantations, Markets)
    - Group Style: Label (14px, gold), items (16-18px)
    - Spacing: 60px padding, 40px gap between groups
- **Right Panel (Visual):**
  - Width: 50%
  - Background: Stylized map or port imagery
  - Map Style: Flat illustration, Indonesia highlighted, connection lines to markets
  - Map Colors: Gold markers, subtle blue/gray paths
- **Responsive:** Mobile: Stacked, map on top or bottom

#### 6. Product Overview Section
- **Type:** Split (50/50), viewport-height
- **Left Panel (Visual):**
  - Width: 50%
  - Background: Image with overlay
  - Image: CPO storage tanks, loading operations, or product container
  - Overlay: Burnt Sienna (#c45b28) at 20% opacity
- **Right Panel (Content):**
  - Width: 50%
  - Background: Warm Cream (#f5f0e8)
  - Color: Dark Coffee (#2c2416) text
  - Content:
    - Section Title: H2, Burnt Sienna (#c45b28), 36-42px
    - Product Name: H3, 24-28px
    - Specs Table: 4-5 rows, label + value format
    - Spec Row Height: 40-50px
    - Quality Notes: 2-3 bullet points, 16px
    - Spacing: 60px padding
- **Responsive:** Mobile: Stacked, image on top

#### 7. Contact Section
- **Type:** Full-width
- **Height:** 80vh minimum (less than full to avoid excessive scroll)
- **Background:** Dark Coffee (#2c2416)
- **Content:** Centered, max-width 800px
- **Elements:**
  - Headline: H2, white, 42-48px
  - Subhead: Body, Warm Cream, 18-20px
  - Contact Grid: 2-3 columns (email, phone, address)
  - CTA Button: Large, gold accent (#d4a24c), 56px height
  - Footer: Copyright, social links, certifications
- **Spacing:** 80px top/bottom padding
- **Responsive:** Mobile: Single column, reduced padding

---

### Visual Consistency Rules (Layout 1)

- **Spacing Scale:** 20px, 30px, 40px, 60px, 80px (multiples of 10)
- **Typography Scale:**
  - H1: 48-64px
  - H2: 36-42px
  - H3: 24-28px
  - Body: 16-20px
  - Caption: 12-14px
- **Border Radius:** 0-4px (industrial feel, no rounded corners)
- **Shadows:** Subtle, 4px blur, 10% opacity (if used)
- **Transitions:** 0.3s ease on hover states

## Layout 2: Asymmetric Dynamic

**Approach:** Varying split ratios (60/40, 40/60, 70/30) and alternating content positions create visual interest and guide attention dynamically.

### Section-by-Section Specifications

#### 1. Hero Section
- **Type:** Full-width
- **Height:** 100vh
- **Background:** Split diagonal or gradient
  - Left 70%: Burnt Sienna (#c45b28)
  - Right 30%: Deep Teal (#2a5c55)
- **Content:** Left-aligned, 80px left padding
- **Elements:** Same as Layout 1 but left-aligned
- **Responsive:** Mobile: Full-width gradient

#### 2. Key Statistics Section
- **Type:** Asymmetric (60/40)
- **Left Panel (60%):**
  - Background: Dark Coffee (#2c2416)
  - Content: Stats grid (2x2), 4 stats
  - Grid: 2 columns, 2 rows, 30px gap
  - Stat Size: Large number (56-72px)
- **Right Panel (40%):**
  - Background: Warm Cream (#f5f0e8)
  - Content: Extended context, 180 words
  - Plus: Small image or icon graphic
- **Responsive:** Mobile: Stacked

#### 3. Sustainability Section
- **Type:** Asymmetric (40/60)
- **Left Panel (40%):**
  - Background: Deep Teal (#2a5c55)
  - Color: Warm Cream text
  - Content: Title + short body (80 words) + certifications row
- **Right Panel (60%):**
  - Background: Image with overlay
  - Image: Larger, more prominent than Layout 1
  - Overlay: Teal, 30% opacity
- **Responsive:** Mobile: Stacked

#### 4. Mission & Vision Section
- **Type:** Asymmetric (50/50) with diagonal split
- **Left Panel:**
  - Background: Image
  - Diagonal clip path for visual interest
- **Right Panel:**
  - Background: Warm Cream
  - Content: Title, Mission block, Vision block, Values grid (2+3 or 3+2)
- **Responsive:** Mobile: Stacked, no diagonal

#### 5. Geographic Presence Section
- **Type:** Asymmetric (55/45)
- **Left Panel (55%):**
  - Background: Stylized map (larger than Layout 1)
  - Interactive elements: Hover markers, connection animations
- **Right Panel (45%):**
  - Background: Warm Cream
  - Content: Location details in card format
  - Card Style: 3 cards, slight shadow, 12px border radius
- **Responsive:** Mobile: Stacked, cards become full-width

#### 6. Product Overview Section
- **Type:** Asymmetric (45/55)
- **Left Panel (45%):**
  - Background: Product imagery, edge-to-edge
- **Right Panel (55%):**
  - Background: Warm Cream
  - Content: Title, detailed specs table (expanded), quality badges
- **Responsive:** Mobile: Stacked

#### 7. Contact Section
- **Type:** Full-width with accent panel
- **Layout:** Left 70% content, right 30% accent
- **Left Panel:** Contact form and info
- **Right Panel:** Deep Teal accent with subtle pattern
- **Responsive:** Mobile: Stacked, accent becomes bottom border

---

### Visual Interest Elements (Layout 2)

- **Diagonal Cuts:** 15-30 degree angles on section transitions
- **Alternating Ratios:** 60/40 → 40/60 → 55/45 → 45/55 (never same twice)
- **Card Components:** Used for grouped information
- **Hover Effects:** Scale (1.02), shadow increase on cards
- **Parallax:** Subtle on images (10-20% scroll speed difference)
- **Micro-animations:** Fade-in on scroll, icon animations

## Layout 3: Card Grid Modular

**Approach:** Content organized into consistent card components within full-width sections, creating modular, scannable layout.

### Section-by-Section Specifications

#### 1. Hero Section
- **Type:** Full-width
- **Height:** 100vh
- **Background:** Carousel or video with overlay
- **Content:** Center card (glassmorphism effect)
  - Card Background: rgba(255,255,255,0.1) backdrop blur
  - Card Padding: 60px
  - Border: 1px solid rgba(255,255,255,0.2)
- **Responsive:** Mobile: Card becomes full-width with larger padding

#### 2. Key Statistics Section
- **Type:** Full-width with grid
- **Background:** Dark Coffee (#2c2416)
- **Content:** 4 cards in 4-column grid
  - Card Style: Light background, 20px padding, 8px border radius
  - Card Size: Equal width, 200px height minimum
  - Stat Number: 56-72px, gold accent
  - Label: 14-16px, below number
- **Spacing:** 40px gap, 60px side padding
- **Responsive:** Mobile: 2x2 grid or 1-column stack

#### 3. Sustainability Section
- **Type:** Split with card integration
- **Left (40%):** Intro text card
- **Right (60%):** Certification badges in card grid (2 columns)
  - Badge Card: 120px square, centered icon, label below
  - Badge Count: 4-5 cards (RSPO, ISCC, Traceable, Carbon, Sustainable %)
- **Responsive:** Mobile: Stacked, cards become 1-column

#### 4. Mission & Vision Section
- **Type:** Full-width with card rows
- **Background:** Warm Cream
- **Content:**
  - Mission Card: Full-width, large text, gold left border
  - Vision Card: Similar to mission, visual distinction
  - Values Grid: 5 cards in responsive grid (3+2 or 5 across)
    - Value Card: Icon top, text below, 160px width
    - Icon Size: 40-48px
    - Text: 14-16px, centered
- **Responsive:** Mobile: Cards stack, grid becomes 1-column

#### 5. Geographic Presence Section
- **Type:** Full-width with card sections
- **Section 1 (25%):** HQ and Office cards (side by side)
- **Section 2 (40%):** Plantation Regions cards (3 across)
- **Section 3 (35%):** Export Markets cards (Primary/Secondary split)
- **Card Style:**
  - Background: White or light cream
  - Border: 1px solid rgba(44,36,22,0.1)
  - Padding: 24px
  - Border Radius: 8px
- **Responsive:** Mobile: All cards stack vertically

#### 6. Product Overview Section
- **Type:** Split with card specs
- **Left (45%):** Product hero image
- **Right (55%):** Specs in card format
  - Main Product Card: Product name, description
  - Spec Cards: 4-5 small cards (2x2 grid)
    - Spec Card: Label top, value bottom, 100px height
  - Quality Cards: 2-3 cards below specs
- **Responsive:** Mobile: Stacked, specs become 1-column

#### 7. Contact Section
- **Type:** Full-width with card center
- **Background:** Dark Coffee
- **Content:** Central card (max-width 600px)
  - Card Background: Slightly lighter than bg
  - Contact Info Cards: 3 inline (email, phone, address)
  - Form Card: Inputs as cards
  - CTA: Large button card
- **Responsive:** Mobile: Central card padding reduces, info cards stack

---

### Card Component Specifications (Layout 3)

| Property | Value |
|----------|-------|
| Background | White (#ffffff) or Warm Cream (#f5f0e8) |
| Border | 1px solid rgba(44, 36, 22, 0.1) |
| Border Radius | 8-12px |
| Padding | 20-32px |
| Shadow | 0 2px 8px rgba(0,0,0,0.08) |
| Hover Effect | TranslateY(-4px), shadow increase |
| Transition | 0.3s ease |

### Grid System (Layout 3)

- **Columns:** 12-column grid for precise alignment
- **Gutters:** 24-32px
- **Max Width:** 1400px with 80px side margins
- **Breakpoints:**
  - Desktop: 4 columns
  - Tablet: 3 columns
  - Mobile: 1-2 columns

## Comparison Summary

| Aspect | Layout 1: Classic Split | Layout 2: Asymmetric | Layout 3: Card Grid |
|--------|------------------------|---------------------|---------------------|
| **Visual Rhythm** | Predictable, orderly | Dynamic, varying | Modular, scannable |
| **Complexity** | Low | Medium | Medium-High |
| **Readability** | Very High | High | High |
| **Portfolio Impact** | Professional, clean | Creative, distinctive | Modern, component-based |
| **Implementation** | Simple | Moderate | Moderate |
| **Best For** | Traditional B2B | Creative showcase | Modern web standards |
| **Maintenance** | Easy | Moderate | Moderate |

## Recommendation

For a portfolio showcasing modern industrial design:

- **Layout 1 (Classic Split)** — Safe, professional, easy to implement
- **Layout 2 (Asymmetric Dynamic)** — Highest creative impact, demonstrates design thinking
- **Layout 3 (Card Grid)** — Modern, component-based, shows modern CSS skills

**My Recommendation:** Layout 2 (Asymmetric Dynamic) for portfolio differentiation while maintaining professional credibility.

---

## Implementation Priority

1. **Core Structure:** HTML skeleton with all sections
2. **CSS Variables:** Define colors, spacing, typography
3. **Layout Grid:** CSS Grid or Flexbox implementation
4. **Responsive Breakpoints:** Mobile-first, tablet, desktop
5. **Interactions:** Scroll behavior, hover states
6. **Polish:** Animations, transitions, micro-interactions
7. **Accessibility:** ARIA labels, focus states, contrast checks
